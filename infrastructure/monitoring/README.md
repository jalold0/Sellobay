# Monitoring Stack

## Komponentlar (rejada)

- **Prometheus** + **Grafana** — metrikalar va dashboard'lar
- **Loki** + **Promtail** — markazlashtirilgan loglar
- **Sentry** — xato kuzatuvi
- **Uptime Kuma** yoki Better Stack — uptime monitoring

## Dashboards

- API: RPS, p95/p99 latency, 5xx error rate
- DB: connection pool, slow queries, lock contention
- Buyurtmalar: per-minute volume, conversion funnel
- Kuryer: yetkazib berish vaqti taqsimoti, muvaffaqiyat darajasi
- Marketing: kampaniya ROI, push CTR
