# @ecom/telegram-mini-app — Telegram Mini App

Telegram WebApp SDK orqali Telegram ichidagi e-commerce ilova (Next.js, port 3003).

Port: `3003`

## Bot bilan ulash

1. BotFather'da bot yarating va `/setmenubutton` orqali Mini App URL berib qo'ying (https://your-domain.uz).
2. `.env`da `TELEGRAM_BOT_TOKEN` ni qo'shing.
3. Production'da HTTPS shart, lokalda ngrok orqali test qilish mumkin.
