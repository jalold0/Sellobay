# @ecom/telegram-bot

grammY asosida Telegram bot. Bot orqali:

- `/start` — boshlang'ich xabar va Mini App tugmasi
- `/shop` — do'konni ochish (WebApp)
- `/orders` — buyurtmalar
- `/help` — yordam
- Marketing kampaniyalari (admin paneldan yuboriladi)
- Buyurtma holati yangilanganda push xabarlari
- Qo'llab-quvvatlash so'rovlari (CRM bilan integratsiya)

`.env`'da `TELEGRAM_BOT_TOKEN` va `TELEGRAM_WEBAPP_URL` ni sozlang.
