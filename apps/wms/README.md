# @ecom/wms — Ombor Boshqaruv Tizimi

Korporativ darajadagi ombor boshqaruv tizimi (NestJS).

## Modullar

- `warehouses` — Omborlar ro'yxati va tafsilotlari
- `inventory` — Stok darajalari, qabul qilish, jo'natish, adjust
- `movements` — Stok harakatlari jurnali (Receiving, Dispatch, Transfer, Adjustment)

## Imkoniyatlar

- Real vaqt inventar kuzatuvi
- Barcode/QR-kod (kelajakda)
- Bir nechta ombor va joylashuv (location)
- Inventarizatsiya (audit)
- Stok harakatlari to'liq jurnali
- Omborlar o'rtasida transfer

Port: `4001`
